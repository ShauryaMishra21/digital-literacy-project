For Task-1, I created a digital literacy awareness infographic using Canva. The aim of the infographic was to present important concepts of digital literacy in a simple, visually appealing, and easy-to-understand format for students.

In my design, I included key sections such as “What is Digital Literacy,” “Useful Digital Tools for Students,” “Safe Internet Practices,” and “Professional Online Presence.” Each section contains short and clear bullet points, supported by icons and a consistent color theme to improve readability and engagement. The infographic highlights essential practices like using strong passwords, avoiding suspicious links, and maintaining a professional online image.

One interesting aspect of this task was learning how to organize information visually instead of writing long paragraphs. I also explored Canva’s design features like templates, icons, and layout adjustments.

Overall, this task helped me understand the importance of digital literacy and how visual tools can be used effectively to spread awareness.
